## ūüĆĪ What is Agro-Mind?

Agro-Mind is an end-to-end computer vision pipeline that classifies crop diseases from leaf images. A farmer uploads a photo ‚ÄĒ the model returns the plant type, disease name, and a confidence score. Built to replace slow, expensive manual diagnosis with a fast, accurate, and always-available AI system.
